# 📊 Data Analysis Python Project

A collection of data analysis and data science projects built with Python and Jupyter Notebooks. This repository showcases end-to-end data workflows — from cleaning and exploration to visualization and predictive modeling — across a diverse range of real-world datasets.

---

## 🗂️ Projects

### 1. 🍵 Cafe Cleaning Data
**File:** `cafe_cleaning_data.ipynb`

Data cleaning and preprocessing project focused on a café dataset. Covers handling missing values, standardizing formats, removing duplicates, and preparing raw data for downstream analysis.

**Key Skills:** `pandas` · Data Wrangling · Missing Value Imputation · Data Quality

---

### 2. 🍫 Chocolate Sales Analysis
**File:** `chocolate_sales_analysis.ipynb`

Exploratory data analysis (EDA) on chocolate sales data. Uncovers trends in sales performance, top-selling products, regional breakdowns, and seasonal patterns.

**Key Skills:** `pandas` · `matplotlib` · `seaborn` · EDA · Sales Analytics

---

### 3. 🥇 Data Science Gold Prediction
**File:** `data_science_gold_prediction_pr....ipynb`

A predictive modeling project aimed at forecasting gold prices using historical market data. Applies machine learning techniques to build and evaluate regression models.

**Key Skills:** `scikit-learn` · Feature Engineering · Regression · Model Evaluation

---

### 4. 👟 Footwear Sales Analysis
**File:** `footware_sales.ipynb`

Analysis of footwear sales data to identify high-performing categories, revenue trends, and customer purchasing behavior across different product lines.

**Key Skills:** `pandas` · `matplotlib` · Sales Insights · Trend Analysis

---

### 5. 🎬 IMDB Final Project
**File:** `imdb_final_project.ipynb`

In-depth analysis of the IMDB movies dataset. Explores ratings, genres, directors, box office performance, and what factors correlate with movie success.

**Key Skills:** `pandas` · `seaborn` · Correlation Analysis · Entertainment Data

---

### 6. 🗾 Japan Population Project
**File:** `japan_population_project.ipynb`

Demographic analysis of Japan's population data over time. Examines population trends, aging statistics, regional distribution, and projections.

**Key Skills:** `pandas` · `matplotlib` · Demographic Analysis · Time Series

---

### 7. 🎬 Netflix Dataset Analysis
**File:** `netflix_dataset.ipynb`

Exploratory analysis of the Netflix content library. Investigates content type distribution, country of origin, genre trends, and growth of the platform over the years.

**Key Skills:** `pandas` · `plotly` / `seaborn` · Content Analytics · EDA

---

### 8. ⚽ Football Dataset (Project 7)
**File:** `project_7_football_dataset.ipynb`

Analysis of football (soccer) statistics including player performance, team comparisons, match outcomes, and league-level insights.

**Key Skills:** `pandas` · `matplotlib` · Sports Analytics · Statistical Analysis

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/Data_Analysis_python_project.git
   cd Data_Analysis_python_project
   ```

2. **Install dependencies**
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn jupyter
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open any `.ipynb` file and run the cells to explore the analysis.
